Display a box About Post Author with author name, avatar and description after each blog post.

== Description ==

This plugin will display a box About Post Author with author name, avatar and description after each blog post.

== Features ==

* Author Avatar
* Author Name
* Author Description
* Change Background Color
* Change Text Color

== Installation ==

Install the plugin via WordPress Dashboard (from plugin directory or from your computer), or upload the files manually to your server and activate the plugin.

= Install via WordPress dashboard =

*From Plugin Directory*

* Visit ‘Plugins > Add New’
* Search for ‘About Post Author’
* Install and Activate the plugin
* You will be redirected to the settings page

*From Your Computer*

* Download the About Post Author plugin
* Visit ‘Plugins > Add New > Upload Plugin’
* Choose the plugin file in .zip format
* Click the Install Now button, and Activate after installation
* You will be redirected to the settings page

= Install via Uploading Files Manually =

* Download the About Post Author plugin and unzip the folder
* Upload the folder to your ‘/wp-content/plugins/’ directory
* Activate the About Post Author plugin from Dashboard
* You will be redirected to the settings page

== Frequently Asked Questions ==

= How to enable author box? =

Activate the plugin and configure the settings. That's it.

= How to disable author box? =

Just deactivate the plugin.
